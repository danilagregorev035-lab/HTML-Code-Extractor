

Extract the source code from any HTML file in seconds

This tool is designed for web developers, designers, and anyone who needs to quickly view or copy the contents of an HTML document. Simply drag and drop the file, and the code will be displayed in front of you.

## ✨ Features

| Function | Description |

| 📂 Uploading files  *Select button or drag-and-drop (Drag & Drop)* 
| 🌍 Two languages | *Russian and English interface*
| 📋 Copying |*Instant copying of code to the clipboard* |
| 🔢 Counting lines | *Automatically detects the number of lines in a file* |


 How to use!!!!


*# Just open index.html in any browser*
*# No servers, builders, or dependencies required*
